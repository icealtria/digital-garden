拓扑排序

拓扑排序（Topological Sorting）是一种对有向无环图（DAG, Directed Acyclic Graph）中节点进行线性排序的方法，使得对于图中的每一条有向边 (u, v)，节点 u 都排在节点 v 的前面。

关键特点

  1. 适用范围：
    • 只适用于 有向无环图（DAG）。
    • 如果图中有环，则无法进行拓扑排序。
  2. 结果：
    • 输出是图中节点的一个线性排列（可能不唯一）。
    • 满足「所有前置节点出现在后置节点之前」。

常见用途

拓扑排序主要用于以下场景：

  • 依赖关系：
    • 任务调度（任务 A 必须先完成，才能进行任务 B）。
    • 软件包依赖（一个软件包依赖另一个包安装）。
  • 解析编译依赖：
    • 编译程序时，先编译依赖的模块。
  • 逻辑依赖：
    • 系统构建和优先级关系处理。

实现方法

通常有两种常见算法实现拓扑排序：

1. Kahn's Algorithm（基于入度）

  • 原理：
    • 从入度为 0 的节点开始处理。
    • 删除节点，并减少其指向节点的入度。
    • 重复此过程，直到所有节点被处理。
  • 步骤：
    1. 计算每个节点的入度。
    2. 将所有入度为 0 的节点加入队列。
    3. 从队列中取出一个节点，将其添加到排序结果中。
    4. 删除该节点，并减少所有指向它的节点的入度。
    5. 如果某节点的入度变为 0，将其加入队列。
    6. 重复步骤 3–5，直到队列为空。
  • 复杂度：O(V + E)，其中 V 是节点数，E 是边数。

2. 深度优先搜索（DFS）

  • 原理：
    • 使用递归，从一个节点开始深度搜索。
    • 当所有后继节点都被访问时，将当前节点加入排序结果。
  • 步骤：
    1. 初始化一个访问状态表，记录每个节点是否已访问。
    2. 遍历每个节点，若未访问则执行以下步骤：
      • 标记节点为「正在访问」。
      • 递归访问所有指向的节点。
      • 标记节点为「已访问」并将其加入排序结果。
    3. 排序结果在递归结束时需要反转，得到正确顺序。
  • 复杂度：O(V + E)。

示例：任务调度

假设有任务依赖关系：

  • A -> B （完成任务 A 后才能做任务 B）。
  • B -> C。
  • A -> D。

拓扑排序步骤：

  1. 构建有向图： 
    A -> B -> C
A -> D
  2. 结果可能是： 
    [A, B, D, C] 或 [A, D, B, C]
    两种排序都满足依赖关系。

注意事项

  1. 如果图中有环，拓扑排序无法完成。
  2. 如果一个节点同时有多个前置依赖，拓扑排序需要优先满足所有依赖。

代码示例

基于 Kahn 算法：

function topologicalSort(graph: Map<number, Set<number>>): number[] {
    const inDegree = new Map<number, number>();
    const result: number[] = [];

    // 初始化入度表
    graph.forEach((edges, node) => {
        if (!inDegree.has(node)) inDegree.set(node, 0);
        edges.forEach(edge => {
            inDegree.set(edge, (inDegree.get(edge) || 0) + 1);
        });
    });

    // 将入度为 0 的节点加入队列
    const queue = [...inDegree.entries()].filter(([_, degree]) => degree === 0).map(([node]) => node);

    while (queue.length > 0) {
        const node = queue.shift()!;
        result.push(node);

        // 更新指向节点的入度
        graph.get(node)?.forEach(neighbor => {
            inDegree.set(neighbor, inDegree.get(neighbor)! - 1);
            if (inDegree.get(neighbor) === 0) queue.push(neighbor);
        });
    }

    return result;
}

基于 DFS：

function topologicalSortDFS(graph: Map<number, Set<number>>): number[] {
    const visited = new Set<number>();
    const result: number[] = [];

    function dfs(node: number) {
        if (visited.has(node)) return;
        visited.add(node);
        graph.get(node)?.forEach(dfs);
        result.push(node);
    }

    // 遍历所有节点
    graph.forEach((_, node) => {
        if (!visited.has(node)) dfs(node);
    });

    return result.reverse(); // 结果反转
}
